# 短视频推荐系统 python1282

本项目是一款基于大数据技术的短视频推荐系统，采用 Django 框架与 Spark 进行开发，旨在为用户提供个性化的短视频内容推荐。

## 技术栈

- **后端框架**：Django
- **编程语言**：Python 3.7/3.8
- **大数据处理**：Spark
- **数据库**：MySQL 5.7 或更高版本
- **数据库工具**：Navicat 11（推荐）
- **开发工具**：PyCharm、VS Code
- **前端框架**：Vue.js

## 功能模块

- 用户管理
- 视频内容管理
- 推荐算法模块
- 数据分析模块

## 系统角色

- 用户：浏览视频，接收推荐内容
- 管理员：管理用户及视频内容，监控系统运行状态

## 运行环境

- Python 3.7/3.8
- Django
- Spark
- MySQL 5.7 或更高版本
- Vue.js

## 部署步骤

1. 环境搭建：安装 Python、Django、Spark、MySQL 和 Vue.js
2. 数据库配置：使用 MySQL 5.7 或更高版本，建立数据库并配置
3. 后端部署：配置 Django 项目，连接数据库，部署后端服务
4. 前端部署：使用 Vue.js 构建前端页面，并与后端服务对接
